## 简介

Vue3 Element Admin 是一个免费开源的中后台模版。使用了最新的`vue3`,`TypeScript`等主流技术开发，开箱即用的中后台前端解决方案，也可用于学习参考。

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/Eaaon/vue3-element-admin.git
```

- 安装依赖

```bash
cd vue3-element-admin

npm install

```

- 运行

```bash
npm run serve
```

- 打包

```bash
npm run build
```
